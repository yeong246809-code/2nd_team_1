# 📍 K-Market에 오신 것을 환영합니다!

> **K-Market은** 한국의 다양한 상품을 한곳에서 만날 수 있는 온라인 쇼핑 플랫폼입니다.
> 편리한 쇼핑 경험과 믿을 수 있는 서비스로 여러분의 생활을 풍요롭게 만들어드립니다.

---

## 🛍️ 주요 기능

| 기능 | 설명 |
|------|------|
| 🔍 **상품 검색** | 카테고리별, 키워드별 상품 검색 |
| 🛒 **장바구니** | 원하는 상품을 장바구니에 담아 구매 준비 |
| 💳 **결제 시스템** | 안전한 결제 처리 및 다양한 결제 수단 |
| 👤 **회원 관리** | 개인정보 관리 및 주문 이력 확인 |
| ⭐ **리뷰 & 평점** | 상품에 대한 솔직한 의견 공유 |
| 📦 **배송 추적** | 실시간 배송 현황 확인 |

## 🚀 시작하기

### 1️⃣ 저장소 복제
```bash
git clone https://github.com/yeong246809-code/2nd_team_1.git
cd 2nd_team_1
```

### 2️⃣ 환경 설정
```bash
npm install      # 의존성 설치
npm start        # 서버 실행
```

### 3️⃣ 데이터베이스 설정
```bash
# 데이터베이스 초기화
mysql -u root -p < database/init.sql
```
